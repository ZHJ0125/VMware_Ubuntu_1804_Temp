                               说明文档

欢迎使用 VMware_Ubuntu_18.04 虚拟机压缩包，系统版本 Ubuntu_18.04_LTS
该压缩包内置了一些开发中常用的软件，包括VSCode、Chrome、搜狗中文输入法等
* 联系方式: https://github.com/ZHJ0125
* 下载链接: https://www.aliyundrive.com/s/RiETdvXSvMb
* 版本: V0.1-20220205
-----------------------------------------------------------------------
文件说明：
├─ README.txt        --> 说明文档
├─ RenameAndHash.bat --> 用于将压缩包的txt后缀纠正为rar，并校验压缩包的哈希值
└─ VMware_Ubuntu1804_Temp.txt --> 虚拟机压缩包
（由于阿里云盘分享限制，故将虚拟机压缩包后缀改为txt，按操作说明运行脚本可将其纠正）
-----------------------------------------------------------------------
操作说明：
0. Git 用户请先移步至阿里云盘下载完整文件夹，下载链接见上文
1. 运行 RenameAndHash.bat 脚本，该脚本会自动纠正文件名并检测校验值
2. 若检测无误，您可以解压 VMware_Ubuntu1804_Temp.rar 压缩包
3. 在 VMware 软件中打开该虚拟机即可使用
4. 虚拟机默认密码: 12345
