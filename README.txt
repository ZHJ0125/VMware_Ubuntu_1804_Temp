欢迎使用 VMware_Ubuntu_18.04 虚拟机压缩包，系统版本 Ubuntu_18.04_LTS
该压缩包内置了一些开发中常用的软件，包括VSCode、Chrome、搜狗中文输入法、终端美化等
* 版本: V0.1-20220205
* 虚拟机压缩包: https://pan.baidu.com/s/1R3tkvY-P3Q_ANoVZ7x70tw?pwd=djw4
* 虚拟机OVA文件：https://pan.baidu.com/s/1ZWIZ1izEI1LURlEYhqeQAw?pwd=8421
* 联系方式: https://github.com/ZHJ0125
--------------------------------------------------------------------------------
文件说明：
├─ README.txt        --> 说明文档
├─ RenameAndHash.bat --> 用于将压缩包的txt后缀纠正为rar，并校验压缩包的哈希值
└─ VMware_Ubuntu1804_Temp.txt --> 虚拟机压缩包
（由于阿里云盘分享限制，现将镜像文件上传至百度云盘，下载镜像请前往百度云盘）
--------------------------------------------------------------------------------
操作说明：
├─ 0. 可以直接下载OVA文件然后导入到虚拟机，或者下载RAR压缩包，压缩包需要做1~3步骤操作：
|  ├─ 1. 先移步至百度云盘下载包含RAR压缩包的完整文件夹，下载链接见上文
|  ├─ 2. 运行文件夹中的 RenameAndHash.bat 脚本，该脚本会自动纠正文件名并检测校验值
|  ├─ 3. 若检测无误，您可以解压 VMware_Ubuntu1804_Temp.rar 压缩包
├─ 4. 在 VMware 软件中直接打开OVA文件或解压该虚拟机压缩包即可使用
└─ 5. 虚拟机默认密码: 12345
